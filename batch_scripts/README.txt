========================================
  LAPTOP CONTROL BATCH SCRIPTS
========================================

This collection contains batch files to control and automate your Windows laptop.

HOW TO USE:
-----------
1. Copy the entire "batch_scripts" folder to your laptop
2. Run "MASTER_CONTROL.bat" for a menu of all options
3. Or run individual .bat files directly

CATEGORIES:
-----------

SYSTEM CONTROL:
  - system_info.bat - Display detailed system information
  - volume_control.bat - Control system volume
  - power_options.bat - Shutdown, restart, sleep, hibernate, lock

FILE MANAGEMENT:
  - search_files.bat - Search for files across drives
  - organize_downloads.bat - Auto-organize downloads folder

NETWORK:
  - network_info.bat - View network configuration
  - wifi_control.bat - Manage WiFi connections

MAINTENANCE:
  - disk_cleanup.bat - Clean temp files and optimize disk
  - process_manager.bat - View and kill processes

APPS:
  - quick_launch.bat - Quick launch common applications

MASTER CONTROL:
  - MASTER_CONTROL.bat - Main menu to access everything

NOTES:
------
- Some scripts require Administrator privileges
- Right-click and "Run as Administrator" if needed
- Modify scripts to suit your needs
- Volume control requires NirCmd (optional)

SAFETY:
-------
- All scripts are safe for your own computer
- Review scripts before running if unsure
- Always backup important data
